# Pranav Patel - Portfolio

Interactive 3D portfolio website showcasing my work as a Software & AI Engineer.

## Features

- **3D Particle System**: Interactive THREE.js particle morphing between different geometries
- **Glitch Text Effects**: Cyberpunk-inspired header with glitch animations
- **Responsive Design**: Fully mobile-optimized with tailored navigation
- **Dynamic Content**: Smooth section transitions with animated content cards
- **Professional Sections**:
  - Home/Core - Introduction and overview
  - Experience - Career timeline with detailed role descriptions
  - Education - Academic background
  - Projects - Showcase of technical projects
  - Skills - Tech stack and capabilities
  - Contact - Professional contact information

## Technologies Used

- **Frontend**: HTML5, CSS3, Tailwind CSS
- **3D Graphics**: THREE.js (r128)
- **Animations**: GSAP (GreenSock Animation Platform)
- **Fonts**: Google Fonts (Rajdhani, Syncopate)
- **Icons**: Font Awesome 6.0

## Local Development

1. Clone this repository
2. Open `index.html` in your browser
3. No build process required - it's pure HTML/CSS/JS!

## Deployment

This site is deployed using GitHub Pages and automatically serves from the `main` branch.
